Software I use

I like cool looking software. I also like productive software. These are my picks for a good middle ground.

I may suggest a mix of both CLI and GUI based programs. If I am using mainly a terminal app, or the TTY, I use the CLI programs, but in a regular desktop I gravitiate generally toward the GUI.

Operating System

Linux Distro - Arch Linux
I love Linux - its fast, looks cool, and you can do anything you want, not restricted by Microsoft's BS. Arch Linux is a barebones Linux distribution that seems hard at first, but is simple to use day-to-day with an open mind.
Desktop Enviroment
I use GNOME since it's simple and looks quite nice when doing day-to-day work, but when coding I like to use Hyprland. You can find my basic Hyprland rice, along with all my other configurations here.

Utilities

Terminal
I use Ghostty cause it's feature-rich, reliable, and looks nice out of the box.
Code Editor
I use VSCodium, an open-source fork of Microsoft's VSCode when creating large projects and Neovim for editing small configuration files. I've always liked the idea of ricing vim like all the tech YouTubers do, but I have no real interest of incorporating it into my workflow.
Web Browser
I use Firefox - it's highly customisable, privacy focused and has all the modern web browser features while still supporting 'Manifest V2' - a web standard that lets adblockers work to their fullest potential.
File Explorer
In a TTY, I use lf, otherwise Nautilus, the default in GNOME, is a great pick.

Office/Productivity

Office Applications
I love LibreOffice - its the best Office suite cause it's not monitored by Microsoft and Onedrive isn't being a pain in the butt.
